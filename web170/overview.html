Client: Giddy Throng | WordPress, Web Design

Overview

This project was for an accelerated introductory WordPress course at Seattle Central College in summer 2014. Students were required to learn WordPress, in-part, by building a custom WordPress theme in eight weeks.

The class had the option of doing a personal site, or working with a client. I chose to work with Giddy Throng to gain more client-facing experience.

Skills used: Illustrator, Photoshop, Dreamweaver, HTML, CSS, Responsive Design, WordPress.
