Client: Nan Nalder | Web Design, Custom WordPress Theme

Overview

Seattle artist Nan Nalder hired me to build a site that she could update by herself. I had worked with Nan previously; she was my client for a web-authoring course when I was attending Seattle Central College. In the course, I designed a site concept and built a few non-responsive static pages.

For this 2016 project, I expanded on the original concept and built a custom responsive WordPress theme.

Client / User Goals

My client’s goals were to provide an engaging interface and information about her:

  • Artwork
  • Upcoming Events
  • Instruction
  • Methods of Contact

Skills used: UX Discovery Exercises, Adobe Illustrator, Adobe Photoshop, HTML5, CSS3, jQuery, Responsive Design, PHP, WordPress.
