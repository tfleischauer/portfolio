Client: Nan Nalder | Web Design, Web Authoring

Overview

This piece is a 2014 class project for Web Authoring II at Seattle Central College.

The focus of this Web Authoring project was to design and develop a site for a client that would pass W3C Markup and CSS Validation Services.

Client / User Goals

My client’s goals were to provide an engaging interface and information about her:

  • Artwork
  • Upcoming Events
  • Instruction
  • Methods of Contact

Skills used: UX Discovery Exercises; Illustrator; Photoshop; Dreamweaver; HTML; CSS; basic jQuery (Hover states, Lightbox 2.)
