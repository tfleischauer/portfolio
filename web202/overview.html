reCreate Outside | UI / UX / UCD / Web Design

Overview

This Seattle Central class project focused on the process to produce a User Interface for an ecommerce site.

The process included discovery of:

  • User Needs
  • Site Objectives
  • Functional Specifications

The discovery phases set the foundation to build artifacts from a User-Centered perspective.

The artifacts are:

  • Information Architecture Diagram
  • Interaction Diagram
  • Wire Frames
  • Visual Designs
  • Protosite

Skills used: UX Discovery, Illustrator, Dreamweaver, HTML, CSS, basic PHP.
